# 第三部分：PLC通信与系统集成

<p class="chapter-lead">视觉系统进入产线后，需要与传感器、PLC、运动机构、剔除装置和HMI共同工作。本部分讨论信号连接、通信方式、时序协调和生产换型，当前连载至第60问。</p>

## 学习目标

- 理解视觉系统与PLC之间的职责边界；
- 能够分析触发、结果回传和通信映射；
- 掌握编码器、运动控制和剔除时序的基本思路；
- 了解HMI参数管理与生产换型流程。
