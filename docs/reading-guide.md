# 阅读指南

## 建议的阅读顺序

初次接触机器视觉的读者，建议从第一部分开始。先弄清视野、分辨率、快门、镜头、景深和照明，再进入算法与系统集成。已有项目经验的读者可以直接进入相关章节，但遇到参数定义或成像条件问题时，仍建议回看第一部分。

## 三条阅读路径

| 读者 | 建议路径 | 阅读重点 |
| --- | --- | --- |
| 学生与初学者 | 第一部分 → 第二部分 → 第三部分 | 概念、公式、参数之间的约束关系 |
| 自动化工程师 | 阅读指南 → 第三部分 → 第一部分相关问题 | 触发、I/O、通信、运动同步与换型 |
| 视觉工程师 | 按项目问题检索，再回到章节导读 | 选型边界、算法条件和系统可靠性 |

## 怎样使用问答页面

左侧导航用于在章节和知识单元之间切换；右侧目录列出当前页面包含的问题。每个主问题下的小节用于解释概念、计算方法、应用条件和常见风险。阅读公式时应同时关注变量定义、单位和适用条件；阅读选型结论时应结合具体视野、精度、速度和环境要求。
