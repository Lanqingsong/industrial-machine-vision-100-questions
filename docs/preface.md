# 前言

机器视觉项目常从一个看似简单的问题开始：能不能把这个缺陷检测出来？真正进入现场后，问题会迅速扩展到视野、精度、节拍、光学条件、触发时序、通信协议、机械动作和长期维护。任何一个环节处理不当，都可能让实验室里有效的方案在产线上失去稳定性。

《工业与机器视觉100问》希望把这些分散在产品手册、工程经验和技术资料中的知识，整理成便于查阅和学习的问答体系。全书从成像硬件出发，逐步进入算法、通信和系统集成。问题的排列大体遵循一套视觉系统从需求分析、方案设计到现场运行的过程。

本书面向正在学习机器视觉的学生，也面向需要参与相机选型、设备联调和项目交付的工程人员。读者可以按章节顺序建立完整认识，也可以从当前项目遇到的问题进入，再沿着相关概念向前补充基础。

书中的科学理论、通用公式和客观事实属于公共知识。作者的工作主要体现在问题体系、内容取舍、具体表达、工程解释和章节编排。写作过程中使用人工智能工具辅助文字校对、语言润色和格式检查；技术判断、资料核验和最终定稿由LanQS（兰青松）负责。
