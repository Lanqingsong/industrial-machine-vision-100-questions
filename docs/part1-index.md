# 第一部分：相机、镜头与成像硬件选型

<p class="chapter-lead">机器视觉首先是一套成像系统。只有让目标特征稳定地进入图像，后续算法才有可靠的数据基础。本部分从工业相机参数开始，依次讨论镜头、照明、三维视觉和特殊成像方案。</p>

## 学习目标

- 根据视野、缺陷尺寸和生产节拍建立相机选型约束；
- 理解焦距、光圈、景深、畸变和远心成像之间的关系；
- 根据材料表面与缺陷特征选择照明方式；
- 了解三维视觉、红外、紫外和智能相机的适用边界。
