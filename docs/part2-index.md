# 第二部分：算法基础

<p class="chapter-lead">算法处理的是成像系统提供的数据。本部分围绕图像预处理、特征提取、定位、测量和结果判定等基础问题展开，重点说明算法成立所需的图像条件及其工程边界。</p>

## 学习目标

- 建立从图像质量到算法稳定性的联系；
- 理解常见处理步骤的目的与适用条件；
- 能够根据任务选择基本算法流程；
- 认识参数调节、误检与漏检之间的关系。
